Assisted mode

          
Safe repairs are automatic: sorting, clamping negative values, conservative swapped-pair fixes, and duplicate-RPM cleanup. Every repair is shown in the input log.

        


        

          
Crashday output

          
Always exports from 0 RPM in uninterrupted 1000-RPM steps. Missing steps are filled by linear interpolation.
